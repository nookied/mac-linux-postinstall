one line. everything broken on a fresh linux install on your macbook — fixed.

install

$ bash -c "$(curl -fsSL https://nookied.github.io/mac-linux-postinstall/install.sh)"

if you have curl (fedora, most desktops)

$ bash -c "$(wget -qO- https://nookied.github.io/mac-linux-postinstall/install.sh)"

if you only have wget (some debian/ubuntu installs)

run as your normal user, not as root. the script asks for sudo when it needs it.

what it does

  • detects your macbook model and linux distro
  • installs the critical drivers automatically (wifi, codecs, function keys)
  • asks you to pick optional must-haves: tlp, mbpfan, flathub, dev-tools, gnome-tweaks
  • tells you when a reboot is needed

known issues

  • facetimehd camera on kernel ≥ 6.15 (fedora 44 ships 6.19): upstream regression — driver loads, captures one frame, then gstreamer/pipewire pipeline freezes.
    // cheese, gnome snapshot → frozen
    // browser-based apps (zoom web, google meet, webcamtests.com) work fine — they bypass gstreamer
    // tracker: patjak/facetimehd#315 — no fix as of may 2026
    // debian 13 (kernel 6.12) and ubuntu 24.04 (6.8) sit before the regression — camera streams live in all apps. targets coming.

before you run it

  • finish your linux install and boot in
  • get online — wired usb-c ethernet or phone tethering (wifi probably won't work yet, that's what this fixes)
  • update first: sudo dnf upgrade --refresh -y && sudo reboot
