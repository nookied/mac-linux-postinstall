mac‑linux‑postinstall

One line. Everything broken on a fresh Linux install on your MacBook — fixed.

Install

Paste one of these into a terminal on your freshly-installed Linux system:

bash -c "$(curl -fsSL https://karolnowacki.github.io/mac-linux-postinstall/install.sh)"

If you have curl (Fedora, most desktops).

bash -c "$(wget -qO- https://karolnowacki.github.io/mac-linux-postinstall/install.sh)"

If you only have wget (some Debian/Ubuntu installs).

Run as your normal user, not as root. The script asks for your sudo password when needed.

What it does

  • Detects your MacBook model and Linux distro
  • Installs the critical drivers automatically (WiFi, codecs, function keys)
  • Asks you to pick optional must-haves: TLP, mbpfan, Flathub, dev tools, GNOME tweaks
  • Tells you when a reboot is needed

Before you run it

  • Finish your Linux install and boot into the new system
  • Get online — wired USB-C ethernet or phone tethering (WiFi probably won't work yet, that's what this script fixes)
  • Run a system update first: sudo dnf upgrade --refresh -y && sudo reboot
