# AGENTS.md

All project info, conventions, architecture, gotchas, and workflow expectations live in **[CLAUDE.md](CLAUDE.md)**.

If you are an AI coding agent working on this repo (Claude, Cursor, Aider, Codex, Copilot, anything else), read [CLAUDE.md](CLAUDE.md) **first** and update it whenever you change something durable about the project.

Also keep [CHANGELOG.md](CHANGELOG.md) current — every behavior-changing commit must add a `## [Unreleased]` entry. This is what stops successive agents running in circles.
